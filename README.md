🎮 遊戲說明

《猜猜我在哪裡》 是一款適合兒童與初學者的趣味配對遊戲。玩家需要觀察畫面中央顯示的圖案，從下方四個選項中找出正確答案。遊戲共有 動物王國、交通工具、水果樂園 三種主題，透過遊戲認識不同的事物，提升觀察力與辨識能力。

📖 遊戲玩法
點擊「開始玩」進入遊戲。
選擇想挑戰的主題。
觀察畫面中央顯示的圖案。
從下方四個選項中選出正確答案。
答對即可進入下一題，答錯則可再次嘗試。
完成所有題目後可獲得星星評價與完成獎勵。
🌟 遊戲特色
三種有趣主題自由選擇。
可愛圖示搭配音效回饋。
答對有鼓勵動畫與音樂效果。
顯示闖關進度，增加學習成就感。
操作簡單，適合幼兒與國小學童使用。
🎯 學習目標

透過互動式配對遊戲，培養玩家的：

圖像辨識能力
專注力與觀察力
記憶力與反應能力
基本認知學習能力
